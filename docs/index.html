Spring 全家桶

采用 Spring Boot 作为后台项目的核心组件，依赖于 Spring 这个大家庭几乎可以完成所有的业务要求。

Vue 驱动

前端使用 Vue + Element-UI，在国内拥有强大的生态圈，利于之前有 JS 基础的同学快速上手。

业务统一

后台所有地市统一使用，前台根据地市要求进行特殊化处理，做到同业务不同展示，有利于项目的后期维护工作。
